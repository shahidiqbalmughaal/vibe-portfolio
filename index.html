My 4 Week Vibe Coding & AI Agents Course Journey

A week by week breakdown of tools, concepts, problems solved, and mini-projects.

Week 1 – Foundations of AI & Web Using Lovable.dev
Tools & Concepts

Focus: Understanding AI basics, prompt structure, and starting with Lovable.dev.

  • Tools: ChatGPT, browser, VS Code, GitHub
  • Concepts: prompts, roles, tokens, HTML structure, CSS basics
  • Problems solved: creating my first AI prompts, building Task Flow Manager using Lovable.dev
  • Mini-projects: Personal Task Manager
Week 2 – Coding Challenges & Mini Apps Using Cursor AI & Android Studio
Code in Action

Focus: using JavaScript for interaction, games, and practical utilities.

  • Tools: HTML, CSS, JavaScript, Bootstrap
  • Concepts: DOM events, game loops, timers, responsive layouts
  • Problems solved: user input handling, score tracking, animation triggers
  • Mini-projects: simple FPS-style shooting game, calculator, to-do list
Week 3 – AI Agents & Workflows Using n8n
Agents & Automation

Focus: Building structured AI agents using n8n that follow steps and return JSON outputs.

  • Tools: AI agents, JSON schema, basic workflow tools
  • Concepts: multi-step reasoning, function-like prompts, context management
  • Problems solved: daily planner agent, email draft generator
  • Mini-projects: “Task Manager Agent” with JSON output, FAQ responder agent
Week 4 – Real World Automations Using Make.com
Final Integration

Focus: Combining AI agents, workflows, using make.com to build complete solutions.

  • Tools: AI agents, workflows, HTML/CSS/JS, GitHub Pages
  • Concepts: end to end automations, UX thinking, documentation
  • Problems solved: repetitive email writing, content drafting, task follow up
  • Mini-projects: This portfolio, plus a final AI automation project
